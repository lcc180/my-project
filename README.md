# my-project

### 特别提醒
请将代码提交到develop分支
master分支用于生产环境，请勿直接提交到master分支

### 环境配置：
安装[nodejs](https://nodejs.org/download/)(最新稳定版即可)

### 安装项目依赖
```
$ npm install -g cnpm
```

### 启动项目
```
$ npm run dev
```

### 生成可部署的代码
```
$ npm run build
```
